# 🎵 Music Player Web Application

A modern and responsive **Music Player** built with a focus on performance, smooth animations, and user-friendly design. This application allows users to play, pause, skip tracks, and enjoy a seamless music experience.

---

## 🚀 Features

### 🎧 Core Functionality
- Play / Pause music
- Next / Previous track navigation
- Seek bar (progress control)
- Volume control
- Display current song details

### 🎨 UI/UX
- Clean and modern interface
- Responsive design (mobile + desktop)
- Smooth transitions and animations
- Interactive controls with hover effects

### ⚡ Performance Optimization
- Lazy loading of assets
- Optimized audio handling
- Efficient rendering
- Minimal load time

### ✨ Advanced Animations
- Smooth transitions between tracks
- Animated progress bar
- Interactive hover effects
- Lightweight animation libraries (GSAP / Framer Motion)

## 🛠️ Tech Stack

**Frontend:**
- HTML5

- JavaScript / React

**Libraries & Tools:**
- GSAP / Framer Motion

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/vishwakarthikeya/codeAlpha_MusicPlayer.git
```

### 2️⃣ Navigate to project folder
```bash
cd music-player
```

### 3️⃣ Install dependencies
```bash
npm install
```

### 4️⃣ Run the project
```bash
npm run dev
```

## 🌐 Usage

- Click on a song to start playing
- Use controls to:
  - ⏯ Play / Pause
  - ⏭ Next / Previous
  - 🎚 Adjust volume
  - ⏩ Seek through the track
